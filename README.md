# Julius Theme

A custom WordPress theme developed by TGN Digital.

## Description

Julius Theme is a custom WordPress theme designed for flexibility and performance.

## Version

1.0.0

## Features

- Custom post types
- AJAX functionality
- Theme customizer integration
- Custom meta boxes
- Responsive design
- Custom admin functionality

## Installation

1. Download the theme files
2. Upload to `/wp-content/themes/` directory
3. Activate the theme through WordPress admin dashboard

## File Structure

- `admin/` - Admin functionality
- `ajax/` - AJAX handlers
- `assets/` - CSS, JavaScript, and images
- `cpt/` - Custom post types
- `customizer/` - Theme customizer settings
- `meta/` - Custom meta boxes
- `template-pages/` - Page templates
- `template-parts/` - Template parts for content display

## Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
